Help

First, select a dictionary with the slider. Larger dictionaries will make more secure passphrases, but some of the words might be obscure and harder to remember.

Next, select the number of words in the phrase. Similarly, using more words will make a more secure passphrase, but it will be harder to remember.

You'll see the strength of those settings below, measured in bits. Each extra bit means it takes twice as many attempts to guess your phrase. The strength you need varies depending on how secure you want things to be, and who you ask.

Then click Generate on the toolbar to generate a passphrase with your chosen settings. That's all there is to it; simply start using your passphrase! If you need another, press the Generate button again.

If you really dislike the generated passphrase, you can press the Generate button to generate another. However, don't keep doing this repeatedly as this reduces the randomness of the final phrase and lowers its security.

FAQ

How do I know this program is trustworthy?

The short answer is, you don't know for definite; there are some decisions that you have to make yourself. However, it is open source, so you're welcome to look through the source code if that helps.

For the truly paranoid, I'd recommend Diceware. It uses physical dice and a paper word list so you don't have to trust anyone.

Are the passphrases truly random?

No, but they use a cryptographically secure generator, so it's about as close as you can get.

What about dictionary attacks?

Some people claim that passphrases are vulnerable to dictionary attacks. It's true that a dictionary attack is the fastest way to crack a passphrase - but all of the calculations for entropy / strength already take dictionary attacks into account. That's the whole point!

How large are the dictionaries?

The tiny dictionary is about 730 words (depending on your language), and each larger dictionary approximately doubles that, up to about 80,000 for the extreme dictionary.

Security Tips

Follow all the usual advice for passwords in general:

Never tell anyone. If you write it down, make sure it's kept safe. Never store it on your computer except in a dedicated password manager. Don't use the same one for several sites.

Randomness

The most important thing to know about passphrases in particular is they only work when they are random.

Human beings are really bad at being random. If you try to think of a passphrase yourself, it may seem random to you, but the chances are that it could be easily guessed. That's why we use computers (or dice) to do it for us.

Along similar lines, if you dislike a particular passphrase or find it difficult to remember, I'd recommend generating another one. But don't keep doing that repeatedly as it reduces the randomness.

Memorising

If you can't remember your password instantly, write it down on a piece of paper and keep it in a safe place. Every few hours, try to remember it, then check. When you're confident you've got it, burn or shred the paper.

If you're using strong encryption, don't forget your passphrase, whatever happens. Your encrypted data will be completely lost.

Password Managers

Passphrases work well combined with password managers. A good strong passphrase can be used as the master password for the password manager, and then the password manager can be used to generate gibberish passwords for each account.
